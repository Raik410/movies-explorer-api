# movies-explorer-api

Задняя (back-end) часть дипломной работы.

## Функциональность 

На данном сайте вы можете посмотреть моё портфолио, а также найти небольшое количество фильмов.

Из стека тут используется: 
- JS
- NodeJS
- express
- mongoDB
